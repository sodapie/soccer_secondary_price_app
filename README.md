# [チケジャム](https://ticketjam.jp/)から情報をスクレイピングするためのツール

***

### App's Link
Here : https://soccersecondarypriceapp-6rpcslmfuwjzhpm6ofr79e.streamlit.app/

### 取得できる情報
- `data_get_dates`: 今日の日付
- `event_dates`: 試合日
- `events`: 試合概要
- `details`: チケットに関する説明
- `prices`: チケット価格
- `amounts`: チケット枚数
- `status`: 取引状況
